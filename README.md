Student Grade Management System

📌 Project Description

This is a full-stack web-based Student Grade Management System. It allows administrators, teachers, and students to manage and view academic records such as grades, courses, and student information.

🚀 Features

- Student registration and management
- Teacher dashboard
- Add, update, and delete grades
- View student performance
- Simple and user-friendly interface

🛠️ Technologies Used

- HTML, CSS, JavaScript (Frontend)
- Node.js / PHP (Backend)
- MySQL / SQL (Database)

📂 Project Structure

frontend/ backend/ database/ README.md

⚙️ Installation

1. Clone the repository
2. Import the database file into MySQL
3. Run backend server
4. Open "index.html" in browser or start server

▶️ How to Run

- Start backend server
- Connect database
- Open project in browser
